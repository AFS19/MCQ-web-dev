# MCQ Quiz — Web & Dev Technologies
An interactive multiple-choice quiz app covering 9 web and development topics, with configurable difficulty and per-topic question selection.
## Topics
| Topic | Questions |
|-------|-----------|
| Laravel | 15 (5 per level) |
| PHP | 15 |
| JavaScript | 15 |
| Python | 15 |
| HTML | 15 |
| CSS | 15 |
| React | 15 |
| POO (OOP) | 15 |
| UML | 15 |
## Features
- 135 questions total across 3 difficulty levels: **Facile / Moyen / Difficile**
- Per-topic question count selector (0–5 questions per topic)
- Live score tracking during the quiz
- Instant feedback with explanations after each answer
- Score breakdown by topic at the end
- Restart quiz or return to home from the results screen
## Usage
No build step required. Just open the file in a browser:
open my-teacher-quizes/quiz.html
## Tech Stack
- Vanilla HTML / CSS / JavaScript
- No dependencies or frameworks
- Single self-contained file
